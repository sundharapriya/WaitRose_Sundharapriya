# WaitRose_Sundharapriya
Data Engineering Project
The Project demonstrates an end-to-end DATA ENGINEERING pepeline build using the WaitRose retail dataset. The main objective is to Collect, Tlean, Transform, and Analyze retail data to generate meaningful business insights. The WaitRose dataset contains Product, Pricing, and Category information from the retail Industry. The Project focused on desigining a scalable dat pipeline to process raw data into Structured ans Analisis-Ready dataset
